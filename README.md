# ☕ Drink OverSeas (DOS)



### 🇰🇷 프로젝트 소개
**Drink OverSeas(DOS)**는 전 세계의 다양한 음료와 문화를 연결하는 프리미엄 음료 큐레이션 플랫폼입니다. 사용자는 엄선된 글로벌 음료를 탐색하고, 자신만의 취향을 발견할 수 있는 사용자 중심의 인터페이스를 경험할 수 있습니다.

이 프로젝트는 개발자로서의 기술적 역량을 입증하기 위해 제작된 **개인 포트폴리오**용 결과물입니다.

- **핵심 기능**: Cloudflare D1 기반 상품 관리, JWT 기반 보안 인증 시스템, 반응형 쇼핑몰 UI.
- **주의 사항**: 본 프로젝트의 모든 소스 코드와 디자인 자산에 대한 **무단 전제 및 복제를 금지**합니다.

---

### 🇺🇸 Project Introduction
**Drink OverSeas (DOS)** is a premium beverage curation platform that connects diverse beverages and cultures from around the world. It provides a user-centric interface where users can explore selected global drinks and discover their personal tastes.

This project is a **personal portfolio** created to demonstrate technical expertise in full-stack web development.

- **Core Features**: Product management based on Cloudflare D1, secure authentication system using JWT, and responsive shopping mall UI.
- **Notice**: **Unauthorized redistribution or reproduction** of all source code and design assets in this project is strictly prohibited.

---

## 🛠 Tech Stack
- **Frontend**: Next.js (App Router), TypeScript, Tailwind CSS
- **Backend**: Cloudflare Workers (OpenNext), D1 Database, Redis (JWT Session)
- **Deployment**: Cloudflare Pages
